## Prerequisites

Before you start, make sure you have **Python 3.x** installed.

## Installation

<Steps>
  <Step title="Clone the Repository">
    Download the game code to your computer. ```bash git clone
    [https://github.com/MSGanesh23/pysnake.git](https://github.com/MSGanesh23/pysnake.git)
    cd pysnake ```
  </Step>
  <Step title="Install Pygame">
    Install the required library using pip. ```bash pip install pygame ```
  </Step>
  <Step title="Run the Game">
    Launch the game window. ```bash python snake.py ```
  </Step>
</Steps>

## Troubleshooting

<Accordion title="Game crashes immediately?">
  Make sure you are running the command inside the `pysnake` folder and that
  `pygame` installed without errors.
</Accordion>
